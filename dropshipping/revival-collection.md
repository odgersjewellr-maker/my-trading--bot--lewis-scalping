# The Revival Collection — fashion expansion plan

*Second collection under the REVIVEЯ brand: "revived" fashion — styles brought
back. Muscles today, classics tomorrow. Added to plan 16 July 2026.*

## Why it fits the brand (and when)

The brand story stretches naturally: REVIVEЯ = things that come back.
Recovery tools revive *you*; the Revival Collection revives *styles*. Store
structure: two collections — **RECOVER** (tools) and **REVIVED** (fashion) —
one brand, one look.

**Timing rule: recovery launches first, fashion joins in October.** Two
reasons: (1) splitting focus at launch is the classic new-store killer — one
hero product, one message, until validated; (2) October = AU spring/summer
warm-up, exactly when hats and sunnies come on-season. The roadmap's
mid-October expansion slot is now the Revival Collection.

## The clothing reality check

| Item type | Verdict | Why |
|-----------|---------|-----|
| **Hats** (bucket, dad caps, terry towelling) | ✅ Best entry | One-size/adjustable = almost no sizing returns; light to ship; retro revival is genuinely trending |
| **Socks** (retro stripe crew) | ✅ Good | Stretchy sizing, cheap, great bundle add-on |
| **Tees / track jackets** | ⚠️ Later | Sizing = 20–40% return rates in apparel; only after the store has volume, and with a printed size chart in cm |
| **Sunglasses (retro frames)** | ⚠️ Careful | On-trend, one-size — but AU requires compliance with the AS/NZS 1067 sunglass standard. Only list if the supplier provides certification |
| **"Vintage" branded gear** | ❌ Never | Anything carrying Nike/adidas/AFL/etc. logos from AliExpress is counterfeit. Retro *style*, never retro *brands* |

## Launch lineup (October)

**Sourced (CJ/AliExpress):**

1. **Terry towelling bucket hat** — peak Aussie summer nostalgia (cricket,
   80s Australiana). ~AU$5–8 cost → AU$30–35. One-size. The hero of the
   collection — this is a TikTok Australiana goldmine.
2. **Corduroy dad cap** — 90s revival staple. ~AU$4–7 → AU$28–32. Adjustable.
3. **Retro stripe crew socks** (3-pack) — ~AU$6–9 → AU$25–29. Bundles with
   everything.

**Print-on-demand (Printful/Printify/Gelato — all have AU print facilities):**

4. **REVIVEЯ dad cap** — embroidered mirrored-R wordmark. POD means zero
   inventory, printed in AU, 2–7 day shipping. ~AU$18–22 cost → AU$45.
5. **"REVIVEЯ" retro-font tee** — only once sizing infrastructure is in place
   (size chart, easy exchanges). POD makes returns cheaper to absorb.

POD note: own-branded merch has real margins only at brand strength — it
sells because people like REVIVEЯ, not because they searched for a cap. Let
the recovery side build the audience first; merch converts followers, not
strangers.
